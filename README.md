## novakit
novakit是一款前端项目脚手架，提供模板下载，代码检查，静态服务器等常用的功能。

### 全局安装
```
npm i -g @nova-kit/core
//or
pnpm add -g @nova-kit/core
```

### 功能

- 通用模板下载
- 代码检查，依赖分析
- 静态服务器，支持本地https
